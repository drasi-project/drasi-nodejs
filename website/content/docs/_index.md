Welcome to the documentation for **`@drasi/lib`**, the Node.js binding that embeds
the [Drasi](https://drasi.io) continuous-query engine directly in your application.

## Where to start

- **[Getting Started](getting-started/)** — install `@drasi/lib` and run your first
  continuous query.
- **[Concepts](concepts/)** — the change-driven model: sources, continuous queries,
  and reactions.
- **[Guides](guides/)** — task-focused walkthroughs: JavaScript sources and
  reactions, native plugins, error handling, and TypeScript types.
- **[API Reference](api/)** — every method on the `Drasi` class.
- **[Examples](examples/)** — runnable scripts and the end-to-end trading demo.
